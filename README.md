## 这是一个转化成字符串图片、视频的工具

### 使用方法
1. 安装PIL `pip pil`
2. 安装ffmpeg
3. 根据需求修改 `intervalPixel` `thumbnailMaxSize`，`srcFile`参数
4. 运行脚本

```shell
# 图片字符化
$ python picture.py
# 视频字符化
$ python video.py
```
